### Como Testar

- Clone o Repositório.
- Entre no diretório 'web' e 'backend' no seu terminal.
- Execute 'yarn install' ou 'npm install' para instalar as dependências.
- Execute 'yarn start' ou 'npm start'.
- Certifique - se de ter o Node JS instalado na sua máquina.
